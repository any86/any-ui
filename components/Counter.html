# Counter

计算器, 常见于购物车中增减商品数量, 支持v-model

# 基本使用

{
    data(){
        return {value: 3};
    }
}

<!-- 默认: 最小值0, 最大值10 -->
<a-count v-model="value"/>

# 高级使用

<!-- 最小值为3, 最大值为5 -->
<a-count :min="3" :max="5" v-model="value"/>

<!-- 每次变化为2 -->
<a-count :step="2" v-model="value"/>

# API

# props
参数说明类型默认值可选值是否必选
value当前显示数字NumberString1-是
min最小值Number1-否
max最大值Number10-否
step每步的变化Number2-否
# events
名称说明回调参数参数类型
input数值变化触发当前值Number
reachMin数值到达min值触发当前值Number
reachMax数值到达max值触发当前值Number
