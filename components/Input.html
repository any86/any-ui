# API

# props
参数说明类型默认值可选值是否必选
value输入值String--是
type用来限制输入内容的格式String-bankCode letter phone number否
hasRemove是否有清空按钮(x图标)Booleantrue-否
vaildateblur触发时, 验证输入的规则Array[]-否
hasWarningDialog验证不匹配的时候, 是否出现提示Booleantrue-否
filterExp过滤指定条件的输入String / RegExp--否
# events
名称说明参数参数类型
input输入文字触发输入内容String
warning发生验证不匹配触发提示语String
focus原生focuseventEvent
blur原生focuseventEvent
keyup原生focuseventEvent
keydown原生focuseventEvent
